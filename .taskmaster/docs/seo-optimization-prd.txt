# Mesa Group Consulting - Comprehensive SEO Optimization PRD

## Project Overview

Mesa Group Consulting operates two websites that require comprehensive SEO optimization and Google Search Console verification:

1. **Main React Site**: https://www.mesagroupconsulting.com (React/Vite SPA on Vercel)

This PRD outlines all tasks required to ensure both sites are properly indexed by Google, have correct SEO configurations, and achieve optimal search visibility.

## Technical Notes

### React Site SEO Implementation

The existing `src/components/SEO.tsx` component uses react-helmet-async:

```tsx
import { Helmet } from 'react-helmet-async';

export const SEO = ({ title, description, canonicalUrl, type = 'website', name = 'Mesa Group Consulting' }: SEOProps) => {
  const DOMAIN = 'https://www.mesagroupconsulting.com';
  const effectiveCanonicalUrl = canonicalUrl 
    ? (canonicalUrl.startsWith('http') ? canonicalUrl : `${DOMAIN}${canonicalUrl}`)
    : window.location.href;

  return (
    <Helmet>
      <title>{title}</title>
      <meta name="description" content={description} />
      <link rel="canonical" href={effectiveCanonicalUrl} />
      <meta property="og:type" content={type} />
      <meta property="og:title" content={title} />
      <meta property="og:description" content={description} />
      <meta property="og:url" content={effectiveCanonicalUrl} />
      <meta property="og:site_name" content={name} />
      <meta name="twitter:card" content="summary_large_image" />
      <meta name="twitter:title" content={title} />
      <meta name="twitter:description" content={description} />
    </Helmet>
  );
};
```

### WordPress Blog SEO

Rank Math SEO is installed and configured with:
- Sitemaps enabled
- Author archives set to noindex
- Social meta configured
- Titles & meta configured

### Sitemap Generation

React site uses `generate-sitemap.js` which runs on build:

```json
"build": "vite build && npm run generate-sitemap"
```
